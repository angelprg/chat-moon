# chat-moons
Chat web.
## Tecnologías
  - Backend
    - Node.js
    - Express
    - Socket.io
  - FrontEnd
    - React (Hooks)
    - Webpack
    - Sass
# Instrucciones de uso
## Desde consola ejecutar lo siguiente:
  - $ git clone https://github.com/angelprg/chat-moon.git
  - $ cd chat-moon
  - $ npm install
  - $ node .
